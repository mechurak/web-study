# 배포 실습 기록

이 챕터의 실습은 새 코드가 아니라 **01~03의 실습 앱을 실제로 배포**하는 것이다.
이 폴더에는 그 과정의 기록을 남긴다:

- `checklist.md` — 배포 전/후 점검 목록 (실무 참고용으로 발전시키기)
- `env-inventory.md` — 스택 전체 환경변수 인벤토리
- `deploy-log.md` — 실제 배포하며 겪은 문제와 해결 기록 (가장 가치 있는 산출물)
- `Dockerfile` 등 배포 설정 파일 사본

## 진행 상황

- [ ] 01 Vercel: Next.js 앱 프로덕션 + 프리뷰 배포
- [ ] 02 환경변수: 인벤토리 작성, 환경별 설정
- [ ] 03 Nest 호스팅: 컨테이너화 + 플랫폼 배포 + 전 구간 연결
- [ ] 04 CI/CD: GitHub Actions PR 파이프라인 + 브랜치 보호
- [ ] 05 모니터링: Sentry + Speed Insights + uptime
- [ ] 06 보안: 보안 헤더 채점 + rate limit + 배포 전 보안 체크리스트
